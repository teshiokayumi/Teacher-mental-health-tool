# 教員こころのサポートツール 運用マニュアル

本ツールは特別なサーバー環境（Node.jsやデータベースなど）を構築することなく、**1つのHTMLファイル（スタンドアロンHTML方式）** で動作するように設計されています。

学校の共有サーバーにファイルを1つ置くだけで、校内の全教職員が安全に利用できる仕組みになっています。

---

## 1. 設置方法（管理者向け）

1. `dist` フォルダ内にある **`index.html`** ファイルをコピーします。
2. コピーしたファイルを、学内の共有サーバーの適切な場所（例：`Z:\共有フォルダ\教職員メンタルヘルス_ツール\index.html`）に配置してください。
   *※ ファイル名はわかりやすい名前（例: `こころのサポートツール.html`）に変更しても問題ありません。*

## 2. 先生方への展開方法

各先生方には、各自のパソコンのデスクトップに**ショートカット**を作成していただき、そこから起動するよう案内してください。

**【ショートカットの作成手順】**
1. 共有サーバーに配置した `index.html`（または `こころのサポートツール.html`）を右クリックします。
2. メニューから「ショートカットの作成」を選択します。
3. 作成されたショートカットを、自分のパソコンの「デスクトップ」に移動させます。
4. 以降は、そのショートカットをダブルクリックするだけでツールがブラウザで起動します。

---

## 3. データの安全性と「自分のデータだけが残る」仕組み

このツールは、教職員の皆様に安心して使っていただくために、安全性を最優先した設計になっています。

* **完全に独立したローカル保存（データ混同なし）**
  皆様が全員「共有サーバー上の同じファイル」を開いたとしても、入力したデータ（業務ログやチェック結果など）は、**「そのファイルを開いている各先生方のパソコン内のブラウザ（LocalStorage）」** にのみ保存されます。
  そのため、A先生の入力内容が、B先生の画面に表示されることは絶対にありません。
* **外部サーバーへの通信なし**
  入力した文章やストレスチェックの履歴は、学校の共有サーバーや外部のクラウド等に送信されることは一切ありません。
* **ブラウザを閉じても復元**
  ブラウザの LocalStorage を利用しているため、翌日に再度ショートカットからツールを開いても、昨日入力したご自身の業務時間やチャットなどがそのまま残っており、続きから利用できます。

> **⚠ 注意点（PCを共有している場合のみ）**
> データは「パソコンのブラウザ単位（およびWindowsのログインユーザー単位）」で保存されます。
> 万が一、1台のパソコン（同じWindowsのユーザーアカウント）を複数の先生で使い回している場合は、先に使った方のデータが見えてしまいます。各自に割り当てられたPCを使用するか、Windowsのアカウントを各自切り替えて利用してください。

---

## 4. 主な機能と使い方

1. **💬 入力タブ（チャット感覚のメモ）**
   * 「今日あったしんどいこと」や「保護者対応の悩み」などを自由に書き込めます。
   * 書き込んだ内容は自動分析され、ストレス傾向が算出されます。
2. **📊 ストレスタブ**
   * 入力内容や業務ログから、5項目のストレススコアが自動計算されます。
   * 限界度合いが高い場合は、相談窓口（スクールカウンセラーや各種ホットライン）のアラートが表示されます。
3. **⏱ 業務ログタブ**
   * 部活、保護者対応、授業準備などの時間を入力し、1週間の業務分布を把握します。
   * 労働時間が多すぎる場合は、ストレスチェックの結果に自動反映されます。
4. **📋 ダッシュボード（管理職向け）**
   * 画面右上の「管理職」ボタンから切り替えることで表示されます。
   * 全体の傾向などを把握する**デモ用ダッシュボード**です（※本ツール単体では全教員のデータを集計する機能は持たないため、あくまで見本としての表示となります）。
